# 🎮 5. MVP Architecture (Simple & Fast)

## 🔧 Backend (Python – Flask, fits your scripting style)

```python
# app.py (Python 2 compatible style)

from flask import Flask, jsonify, request
import random

app = Flask(__name__)

# simple board mapping
LADDERS = {3:22, 8:26, 15:44}
SNAKES = {17:5, 24:6}

@app.route("/roll", methods=["GET"])
def roll():
    return jsonify({"dice": random.randint(1,6)})

@app.route("/move", methods=["POST"])
def move():
    data = request.json
    pos = data["position"]
    dice = data["dice"]

    new_pos = pos + dice

    if new_pos in LADDERS:
        new_pos = LADDERS[new_pos]
        event = "ladder"
    elif new_pos in SNAKES:
        new_pos = SNAKES[new_pos]
        event = "snake"
    else:
        event = "normal"

    return jsonify({"new_position": new_pos, "event": event})

if __name__ == "__main__":
    app.run(debug=True)
```

## 🎨 Frontend (Simple HTML + JS)

```html
<!DOCTYPE html>
<html>
<head>
  <title>Gyan Chaupar</title>
</head>
<body>

<h2>Gyan Chaupar Game</h2>

<button onclick="rollDice()">Roll Dice</button>
<p id="output"></p>

<script>
let position = 1;

function rollDice() {
  fetch('/roll')
  .then(res => res.json())
  .then(data => {
    let dice = data.dice;

    fetch('/move', {
      method: 'POST',
      headers: {'Content-Type': 'application/json'},
      body: JSON.stringify({position: position, dice: dice})
    })
    .then(res => res.json())
    .then(res => {
      position = res.new_position;
      document.getElementById("output").innerText =
        "Dice: " + dice +
        " | Position: " + position +
        " | Event: " + res.event;
    });
  });
}
</script>

</body>
</html>
```
